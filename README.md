-Trawa i krak
-Żona i ciąża z obcym
-Panna
-Laptop
-Pole i nasiona
-Hot Dog
-Opieka medyczna
-Karta usług zniżki
-Auto używane
-Brak wypłaty i dużych zarobków do pod wynajem.

Ten system to komunizm wojskowy.
